Intake — understand intent
3 skills

Requirements analyst
Decomposes vague intent into structured specs with acceptance criteria
Domain modeler
Extracts entities, relationships, and invariants from business context
Research scout

Blueprint — architect the system
3 skills
▶
System architect
Selects patterns, defines service boundaries, and produces architecture decision records
API designer
Defines contracts, schemas, and protocol choices before a line of code exists
Data architect
Designs schemas, migration strategies, and data flow topologies
2
Forge — write the code
4 skills
▶
Implementation engineer
Writes production code — backend logic, algorithms, data transformations
Frontend builder
Produces UI components, layouts, and interaction logic from design specs
Integration weaver
Connects services, third-party APIs, message queues, and data pipelines
Migration specialist
Handles database migrations, data transformations, and legacy system bridges
3
Temper — harden and verify
4 skills
▶
Test engineer
Generates unit, integration, and e2e tests with meaningful coverage analysis
Code critic
Reviews for anti-patterns, readability, naming, and architectural drift
Security auditor
Scans for OWASP vulnerabilities, supply chain risks, and secret leakage
Performance analyst
Profiles bottlenecks, benchmarks critical paths, and validates resource usage
4
Finish — polish and ship
4 skills
▶
Documentation writer
Generates READMEs, API docs, architecture guides, and inline comments
DevOps engineer
Produces Dockerfiles, CI/CD pipelines, IaC templates, and deploy scripts
Release manager
Handles versioning, changelogs, canary rollouts, and rollback strategies
Accessibility auditor
Validates WCAG compliance, screen reader compatibility, keyboard navigation
5
Evolve — observe and adapt
3 skills
▶
Observability agent
Instruments telemetry, sets up alerting, interprets logs and traces
Bug triager
Reproduces issues, isolates root cause, and proposes targeted fixes
Refactoring surgeon
Pays down tech debt, modernizes patterns, and migrates across versions
Cross-cutting — skills that span every phase
Orchestrator — decomposes, delegates, synthesizes

Compliance guardian — enforces license, regulatory, and policy rules
